## Cenas

### Cenas recriadas
O projeto possui duas cenas: a primeira é uma reprodução da imagem "01__20260610_223325.png" e a segunda é uma reprodução da imagem "05_20260610_223826.png". Ambas as imagens estão na pasta "cenas-disponiveis" para conferência.

### Navegação entre cenas
Ao executar o projeto, já será possível visualizar as duas cenas: a primeira cena ao lado esquerdo e a segunda cena ao lado direito.
No canto superior esquerdo está disponível um menu que permite alternar entre a visualização das duas cenas com a tela dividida ou a visualização apenas da Cena 1 ou da Cena 2.

## Instalação

### Clonar o projeto

    git clone https://github.com/alanscheibler/trabalho-three.js-g2

### Para executar

- Windows / Linux / MacOS

  No terminal:
    - Acessar a pasta do projeto

          cd trabalho-three.js-g2

    - Executar o comando

          python -m http.server -b 127.0.0.1 8000

    - Acessar via navegador: [http://127.0.0.1:8000](http://127.0.0.1:8000)
